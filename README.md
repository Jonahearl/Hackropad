# Specifications
BOM
- 10x Cherry MX Switches
- 14x SK6812 MINI Leds
- 1x XIAO RP2040
- 10x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset

Others:
- KMK Firmware
- Cover.step
- Base.step
